

 AI-Powered Web UI Testing Framework

This project is an AI-based Selenium automation framework that locates web elements using semantic meaning instead of hardcoded locators.

 🚀 Features
- AI-based element detection
- Uses Sentence Transformers
- Cosine similarity matching
- JSON-based locator storage
- Smart autofill and interaction

 🛠 Tech Stack
- Python
- Selenium WebDriver
- Scikit-learn
- Sentence Transformers

 ⚙️ How it works
1. Scan webpage DOM
2. Store elements in JSON
3. Convert query + elements into embeddings
4. Match using cosine similarity
5. Perform actions (click, input)
